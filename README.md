# Codnixy AI Trade — форекс-ассистент

Приложение поддержки торговых решений. Решения принимают детерминированные
формулы; ИИ-агенты превращают информацию (новости, журнал, опыт) в
ограниченные числовые векторы и объяснения. **Вся отчётность — в евро (EUR).**
Не финансовый совет.

## Архитектура

```
frontend/  Next.js 16 + Tailwind 4 + shadcn/ui + lightweight-charts (порт 3001)
backend/   FastAPI + numpy + SQLAlchemy/SQLite                      (порт 8000)
```

## Данные — Twelve Data (основной провайдер)

REST `time_series` + WebSocket поток цен. Один провайдер покрывает все
категории каталога: форекс (56 пар G8), металлы, индексы, энергоносители,
акции (~205), ETF, криптовалюты (~90). Особенности интеграции:

- **Автовыбор источника** (`auto`): Twelve Data → **EODHD** → OANDA (legacy)
  → симулятор. Символ или таймфрейм, который провайдер не отдаёт (фьючерсы,
  intraday на бесплатном EODHD и т.п.), тихо деградирует к следующему
  источнику — данные есть всегда.
- **EODHD** (дополнительный ключ): дневные свечи включая индексы
  (GSPC/NDX/DJI/DAX/FTSE/N225…) + пакетные котировки real-time. На бесплатном
  тарифе — только EOD, 20 запросов/день (жёсткий кэш 6ч, intraday-блокировка
  определяется автоматически и не тратит лимит).
- **Бюджет запросов**: глобальный лимит в минуту; массовые задачи (скринер,
  тепловая карта, рейтинг волатильности) берут только свободный остаток и
  падают в кэш/симуляцию — графики, сигналы и трекинг всегда в приоритете.
- **WebSocket**: живые котировки избранного (если тариф TD включает WS);
  иначе REST-опрос + кэш. Эндпоинт `/api/quotes`.
- Экономический календарь — ForexFactory (бесплатно), новости — RSS → ИИ.
- Таймфреймы: **5m, 15m, 40m (ресемплинг из 5m), 1h, 4h, 1d**.

### Стакан и глубина (вкладка «Стакан»)
Реального L2 для форекса розничные провайдеры не отдают, поэтому панель
честно разделяет слои: **профиль объёма** (реальные данные из свечей, доля
покупок, три крупнейших уровня «крупных заявок»), **синтетический стакан**
(детерминированная оценка ликвидности от волатильности, профиля и зон S/R,
помечен бейджем «оценка»), **анализ спреда** (оценка в пипсах, стоимость
раунд-трипа 1 лота в EUR, доля спреда в ATR) и дисбаланс bid/ask.

## Конвейер анализа

1. **Индикаторы** (чистый numpy): EMA 20/50, RSI 14, MACD, Bollinger + %B,
   ATR 14, Stochastic, ADX, ROC, VWAP, Ichimoku, Time-series momentum
   (Moskowitz–Ooi–Pedersen 2012), Kaufman ER (1995), Хёрст R/S (Lo 1991).
2. **ИИ-конвейер по расписанию** (1–2×/день, ~3 вызова): Haiku — отбор RSS;
   Sonnet — дебаты быков/медведей → вектор настроений G8; Sonnet — уклон по
   парам. **В промпты подмешивается ИИ-память** (уроки прошлых сделок).
3. **Совокупная оценка** — взвешенная сумма факторов в [−1, 1], доля ИИ
   ограничена (по умолчанию 15%).
4. **Риск-менеджер**: пороги, режим рынка, лимиты, размер позиции (fixed / ½
   Келли) — **в EUR** с конверсией котируемой валюты по живым курсам.

## Возможности

### ИИ-память (постоянное обучение)
- Каждый закрытый сигнал → разбор (какие факторы были «за/против», результат в R).
- Hit-rate каждого фактора и режимов рынка — копится в БД.
- Sonnet периодически «консолидирует» разборы в короткие уроки.
- Память подмешивается во все ИИ-промпты (новости, чат, журнал) — советы
  становятся точнее с опытом. Вкладка «ИИ-аналитика» → панель памяти
  (добавить свою заметку / удалить / извлечь уроки сейчас).

### ИИ-ассистент
- **Чат**: «почему двигался EUR/USD?», «какие риски у сделки?» — контекст:
  техника, паттерны, новости, календарь, портфель, журнал, память.
- **Новости по символу**: релевантные заголовки, тональность
  (positive/neutral/negative), важность, «почему это важно», ранжирование.
- **Разбор журнала**: сильные стороны, ошибки, рекомендации (сохраняются в память).

### Распознавание паттернов (детерминированное, numpy)
Треугольники (симм./восх./нисх.), клинья, каналы, флаги/вымпелы, двойные
вершина/дно, голова-и-плечи (+ перевёрнутая), зоны S/R (кластеризация
пивотов), трендовые линии, пробои и ложные пробои, уровни Фибоначчи.
Каждый паттерн — с объяснением и отрисовкой на графике.

### Риск-менеджмент
- **Калькулятор позиции**: баланс, риск %, вход/стоп, плечо, комиссия, спред,
  ATR-предупреждения → объём, лоты, макс. убыток, маржа, номинал.
- **Умные SL/TP**: трейлинг по ATR, перенос в безубыток при +N R, частичная
  фиксация (доля настраивается). Детерминированный проход по барам,
  стоп проверяется раньше тейка (консервативно).
- **Лимиты**: дневной убыток (€), число убыточных сделок, дневная цель,
  недельный/месячный лимиты, макс. просадка, суммарный открытый риск —
  при достижении новые сигналы блокируются.
- **Риск-монитор**: плавающий P&L по живым котировкам, превышение планового
  риска, коррелированные позиции (одна валюта в одну сторону), приближение к
  лимитам — каждое предупреждение с корректирующим действием.

### Журнал
Стратегия и заметки по каждой сделке; статистика: win rate, profit factor,
матожидание, средние прибыль/убыток, фактический R:R, макс. просадка, серии,
лучший/худший день, разрезы по стратегиям / инструментам / сессиям
(Азия–Лондон–НЙ) / дням недели + ИИ-разбор.

### Алерты
Цена выше/ниже, движение N%, RSI, MACD-кросс, EMA-кросс, пробой Боллинджера,
всплеск ATR/объёма, ИИ-сигнал. Cooldown, заметки. Доставка: приложение
(колокольчик), Telegram, **e-mail (SMTP настраивается в «Подключениях»)**.
Проверка раз в минуту в фоне. За 30 мин до важных новостей — предупреждение.

### Скринер и тепловые карты
Скринер по категориям: изменение 24ч/5д, ATR%, RSI, ADX, тренд, объём,
пробои, моментум-рейтинг; фильтры и сортировка, клик — открыть инструмент.
Тепловые карты всех категорий + **матрица силы валют G8**.

### Бэктест
Тот же движок на истории (до 5000 баров): спред, проскальзывание, комиссия,
компаундинг капитала; SL раньше TP внутри бара; ИИ-факторы честно выключены.
Метрики (WR, PF, матожидание в R и €, Sharpe, просадка), кривая капитала,
список сделок, **walk-forward** (реоптимизация порога на train, оценка на
test), сравнение прогонов, ИИ-критика результатов.

### Графики
Свечи + EMA 20/50, Bollinger, VWAP, Ichimoku, зоны S/R, трендовые линии,
паттерны, уровни Фибоначчи (тумблеры), подокна RSI / MACD / Stochastic,
вход/SL/TP пунктиром, 6 таймфреймов, живая котировка.
**Рисование вручную**: трендлинии (2 клика) и горизонтальные уровни (1 клик),
отмена последней / очистка; сохраняются в браузере по символу+таймфрейму.
**Мульти-график** (вкладка «Мульти»): раскладка 1 / 2 / 2×2, у каждого
графика свой символ и таймфрейм, конфигурация сохраняется.

## Вход и пользователи

- Все API-маршруты защищены; вход обязателен. Первый запуск создаёт
  администратора **admin / admin12345** — смените пароль в «Аккаунт».
- Роли admin/user, журнал действий (входы, настройки, сигналы, IP).
- **2FA (TOTP)**; пароли — scrypt; сессии — токены на 7 дней.

## Быстрый старт

```bash
# backend
cd backend
python3 -m venv .venv && .venv/bin/pip install -r requirements.txt
.venv/bin/uvicorn app.main:app --port 8000

# frontend (другой терминал)
cd frontend
npm install
npm run dev                  # http://localhost:3001
```

Работает сразу: **симуляция данных**, ИИ выключен (нейтральные векторы).

### Подключения — прямо в приложении

Кнопка **«Подключения»**: ключи **Twelve Data** / OANDA (legacy) / Anthropic /
Telegram / SMTP вводятся в UI и хранятся в локальной БД (маскируются).
Там же — источник данных, WebSocket-поток, ИИ-память, расписание анализа,
автосканирование, e-mail для алертов. Расход Anthropic API виден в карточке
(**в EUR**). Переменные окружения в `backend/.env` — значения по умолчанию
(`TWELVEDATA_API_KEY`, `OANDA_API_KEY`, `ANTHROPIC_API_KEY`, …).
